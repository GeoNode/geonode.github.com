Get Started Now

    Install from PPA in Ubuntu 14.04

    $ sudo add-apt-repository ppa:geonode/testing

    $ sudo apt-get update

    $ sudo apt-get install geonode
  • 2
    Set the IP address and create a superuser

    $ sudo geonode-updateip 127.0.0.1

    $ geonode createsuperuser
