GeoNode

Install GeoNode Now Try the Demo

Open Source Geospatial Content Management System

GeoNode is an easy to use application and platform for developing web based Geospatial Information Systems and for deploying interoperable Spatial Data Infrastructure.

It is also designed to be a flexible platform that software developers can extend, modify or integrate against to meet requirements in their own applications.

For Users

  • Browse and search for geospatial data layers and web services
  • Upload, manage and share your own geospatial data
  • Create and share interactive maps
  • Collaborate with others using layers and maps

For Developers

  • Developed with Django
  • Built on stable, mature open source geospatial projects
  • Twitter Bootstrap and jQuery client implementation
  • Multiple APIs for customization and integration
  • Seamless integration with third-party Django apps

For Admins

  • Complete framework for OGC compliant geospatial web services
  • Interoperable metadata managment and aggregation platform
  • Comprehensive security model and framework
  • Reliable and scalable platform for deploying spatial data infrastructure

Quick Start

Get Started Now

  • 1
    Install from PPA in Ubuntu 12.04

    $ sudo add-apt-repository ppa:geonode/snapshots

    $ sudo apt-get update

    $ sudo apt-get install geonode
  • 2
    Set the IP address and create a superuser

    $ sudo geonode-updateip 127.0.0.1

    $ geonode createsuperuser
