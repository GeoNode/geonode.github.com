GeoNode 1.0-beta Release

I'm proud to announce that GeoNode has released its 1.0-beta version. You can build it from source or get the release bundle.

This release includes user management, data upload, metadata management, GeoNetwork-powered data search, and an integrated styling and map composing tool.

Check out the live demo. We will be updating it periodically with bug fixes over the next several weeks, so if something doesn't work as expected now, be sure to revisit soon.

Bug reports are very welcome. Join our public mailing list by emailing geonode@librelist.com or use our public issue tracker.
